Opening the developer tools or resizing them will throw an error in the console

Reproduction:

  1. Open the browsers developer tools (F12).
  2. In the console tab in FireFox you can see this error: 'Uncaught TypeError: r is undefined'.
    In Chromium browswers the error reads: 'Uncaught TypeError: Cannot set property 'more_btn' of undefined'
  3. This will also occur when you already have opened the developer tools and resize the developer tools window.

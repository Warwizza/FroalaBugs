Dragging a block header or footer into another paragraph breaks the styling and loses html tags

Reproduction

  1. Click the question mark button in the toolbar to add the test markup into the editor.
  2. Select the text of the header or the footer of the block (Block for users with an account). Make sure you select the spaces before and after the text to.
  3. Drag the selected text into the other text above or below the block.
  4. The markup of the block will disappear.

Dynamically updating the toolbar is not possible after Froala version 3.2.0

Reproduction:

  1. Click the button below the editor. This should add a button to the Froala editors toolbar
  2. Open the browsers developer tools (F12).
  3. In the console tab you can see that the underline button is added to the the toolbarButtons array.
  4. If you downgrade the Froala version to 3.2.0 or lower and test this again, you will see that the button will be added. You can downgrade by executing the following command in a terminal window: 'npm install angular-froala-wysiwyg@3.2.0''

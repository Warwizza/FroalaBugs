The button toolbar has no highlighting of active buttons.

Reproduction

  1. Type some text in the editor.
  2. Select all or part of the text.
  3. Click the bold button (or press Ctrl + B / Cmd + B).
  4. The bold button is not highlighted. The fr-active class gets applied to the button, but this does not do anything.
  5. Go back to the code and comment or remove the "this.setMaterialSymbolsIcons()" line in the ngOnInit() function (line 25).
  6. Repeat the steps above and now the button is highlighted.

The type of lists doesn't matter. It will remove the line break when you use two ordered lists, two unordered lists or when you mix them.

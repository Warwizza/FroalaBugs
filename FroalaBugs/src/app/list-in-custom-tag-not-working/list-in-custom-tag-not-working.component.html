Creating a list (ordered or unordered) inside a custom tag leads to some strange behaviour.

Reproduction 1

  1. Click the question mark button in the toolbar to add the test markup into the editor.
  2. Click on the text between the block delimiters.
  3. Click the ordered or unordered list button.
  4. The bullet point (or number) will appear before the block delimter.

Reproduction 2

  1. Click the star button in the toolbar to add the test markup into the editor.
  2. Click on the text between the block delimiters.
  3. Click the ordered or unordered list button.
  4. The bullet point (or number) will appear before the text above the block delimiters.

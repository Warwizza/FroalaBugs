Pasting removes line break between lists when the ENTER_BR option is set

Reproduction

  1. Click the ordered list button in the toolbar.
  2. Create a list of two items.
  3. Hit the enter key twice to stop adding items to the list.
  4. Hit the enter key again to create an extra line break.
  5. Click the unordered list button.
  6. Create a list of two items.
  7. Select and cut all the contents of the editor.
  8. Now paste it back in the editor.
  9. The extra line break has been removed.

The type of lists doesn't matter. It will remove the line break when you use two ordered lists, two unordered lists or when you mix them.
